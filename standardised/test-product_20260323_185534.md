# Simple Todo App

## Product Overview
A mobile application that helps individuals organise and track their daily tasks. It provides a straightforward way to create, manage, and complete personal to-dos without the complexity of team collaboration features.

## Problem Statement
Individuals need a simple, focused way to keep track of daily tasks without the overhead of complex project management tools. Existing solutions often include unnecessary features that complicate basic task management.

## Target Users
- **Individual User**: Needs to quickly capture tasks, track due dates, and mark items complete to stay organised in daily life

## Core Features

### Task Management
- Create a new task with a title and due date
- Mark a task as complete
- Delete a task

### Task Viewing
- View all tasks in a list format
- Filter tasks by completion status (complete/incomplete)

## Non-Functional Requirements
- Platform: Mobile (iOS and Android) [Assumed: both platforms based on "mobile app" without specification]
- Tasks should persist between app sessions [Assumed: local storage required for basic functionality]

## Out of Scope
- Team sharing and collaboration
- Push notifications and reminders
