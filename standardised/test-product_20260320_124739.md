# Simple Todo App

## Product Overview
A mobile application designed for individuals to manage their daily tasks efficiently. The app provides a straightforward interface for creating, tracking, and completing personal to-do items, helping users stay organised and productive.

## Problem Statement
Individuals often struggle to keep track of daily tasks and commitments, leading to missed deadlines and forgotten responsibilities. This app provides a simple, distraction-free solution for personal task management without the complexity of team collaboration tools.

## Target Users
- **Individual User**: Needs a quick and easy way to capture tasks, set due dates, and track completion status throughout their day

## Core Features

### Task Management
- Create a new task with a title and due date
- Mark an existing task as complete
- Delete a task from the list

### Task Viewing
- View all tasks in a list format
- Filter tasks by completion status (complete/incomplete)
