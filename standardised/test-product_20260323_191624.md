# Simple Todo App

## Game Overview
A streamlined mobile productivity application that enables users to capture, organise, and complete daily tasks through an intuitive list-based interface. The core loop centers on rapid task creation, satisfying completion interactions, and clear visual progress tracking. Designed for iOS and Android with a focus on speed and simplicity over feature bloat.

## Problem / Opportunity
Existing todo apps suffer from feature creep—calendars, projects, tags, collaboration—creating friction for users who simply want to jot down tasks and check them off. This app targets the "quick capture" moment: the 3-second window when someone thinks "I need to remember to do X" and needs zero-friction task entry. The opportunity is a focused tool that does one thing exceptionally well.

## Target Players
- **Busy Professional (25-45)**: Limited time, uses phone constantly, wants to dump tasks from their head without learning a complex system. Values speed over features.
- **Student (18-24)**: Manages assignments, errands, and personal tasks. Needs simple organisation without the overhead of full project management. Motivated by visible progress.

## Game Design

### Core Gameplay Loop
1. User opens app → immediately sees task list (no login gate, no splash screen)
2. Taps "+" or quick-add field → enters task title and optional due date
3. Task appears in list, sorted by due date
4. User completes task → taps checkbox → satisfying completion animation
5. Completed tasks move to "Done" section or disappear based on filter
6. Repeat throughout day as tasks arise

### Game Mechanics
- **Quick Add**: Single text field always visible at top of list; tap, type, done in under 3 seconds
- **Due Date Picker**: Optional date selection with smart defaults (Today, Tomorrow, This Week, Custom)
- **Swipe Actions**: Swipe right to complete, swipe left to delete (with undo snackbar for 5 seconds)
- **Completion Feedback**: Checkbox fills with micro-animation (0.2s), subtle haptic feedback, task fades/strikes through
- **Filter Toggle**: Segmented control (All / Active / Completed) persists between sessions

### Progression System
- **Task Completion Counter**: Running total of tasks completed (lifetime and today)
- **Streak Tracking**: [Assumed: Days in a row with at least one task completed—displayed subtly, not gamified aggressively]
- **Empty State Celebration**: When all tasks complete, display encouraging message ("All done! 🎉")

### Win / Lose Conditions
- **Win State**: All active tasks completed; empty inbox achievement moment
- **Fail State**: None—overdue tasks visually flagged (red date) but no punishment mechanics; app remains encouraging
