# Simple Todo App

## Product Overview
A mobile application designed for individuals to manage their daily tasks efficiently. The app provides a straightforward interface for creating, tracking, and completing personal to-do items, helping users stay organised without unnecessary complexity.

## Problem Statement
Individuals struggle to keep track of daily tasks and responsibilities, often relying on scattered notes, memory, or overly complex productivity tools. This app provides a simple, focused solution for personal task management that reduces cognitive load and helps users accomplish their daily goals.

## Target Users
- **Individual User**: Needs a simple way to capture tasks as they arise, track due dates, and maintain visibility over what needs to be done. Uses the product to stay organised and ensure nothing falls through the cracks.

## Core Features

### Task Management
- Create a new task with a title and optional due date
- Mark a task as complete to track progress
- Delete a task to remove items no longer needed
- Edit existing tasks [Assumed: users will need to correct mistakes or update task details]

### Task Viewing
- View all tasks in a single list format
- Filter tasks to show only incomplete items
- Filter tasks to show only completed items
