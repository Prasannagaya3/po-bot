# Simple Todo App

## Product Overview
A mobile application that helps individuals manage their daily tasks efficiently. It provides a straightforward way to create, track, and complete personal to-do items, solving the problem of task organisation for busy individuals who need a simple, no-frills productivity tool.

## Problem Statement
Individuals often struggle to keep track of daily tasks and responsibilities, leading to forgotten commitments and reduced productivity. This app provides a lightweight, focused solution for personal task management without the complexity of full project management tools.

## Target Users
- **Individual User**: Needs a simple way to capture, organise, and complete daily tasks; uses the product to stay on top of personal responsibilities and feel a sense of accomplishment when completing items

## Core Features

### Task Management
- Create a new task with a title and optional due date
- Mark a task as complete
- Delete a task from the list
- Edit an existing task's title or due date [Assumed: users need to correct mistakes]

### Task Viewing
- View all tasks in a single list view
- Filter tasks to show only complete items
- Filter tasks to show only incomplete items
- Visual distinction between complete and incomplete tasks [Assumed: for quick scanning]

## Non-Functional Requirements
- Platform: Mobile (iOS and/or Android) [Assumed: "mobile app" implies native or cross-platform mobile]
- Performance: App should load task list within 2 seconds [Assumed: standard mobile performance expectation]
- Data persistence: Tasks should persist between app sessions [Assumed: core expectation for a todo app]

## Out of Scope
- Team sharing or collaboration features
- Push notification reminders
- Recurring tasks
- Categories or tags [Assumed: not mentioned as a feature]
