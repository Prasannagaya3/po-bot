# Simple Todo App

## Product Overview
A mobile application designed for individuals to manage their daily tasks efficiently. The app provides a straightforward way to create, track, and complete personal to-do items, helping users stay organised without unnecessary complexity.

## Problem Statement
Individuals often struggle to keep track of daily tasks and responsibilities, leading to missed deadlines and forgotten commitments. This app provides a simple, focused solution for personal task management without the overhead of complex project management tools.

## Target Users
- **Individual User**: Needs a quick and easy way to capture tasks, track due dates, and mark items as complete to stay on top of daily responsibilities

## Core Features

### Task Management
- Create a new task with a title and due date
- Mark an existing task as complete
- Delete a task from the list

### Task Viewing
- View all tasks in a list format
- Filter tasks to show only complete items
- Filter tasks to show only incomplete items
