# Simple Todo App

## Product Overview
A mobile application designed for individuals to manage their daily tasks efficiently. The app provides a straightforward way to create, track, and complete personal to-do items, helping users stay organised without unnecessary complexity.

## Problem Statement
Individuals often struggle to keep track of daily tasks and responsibilities, leading to missed deadlines and forgotten commitments. This app provides a simple, distraction-free solution for personal task management without the overhead of complex project management tools.

## Target Users
- **Individual User**: Needs a quick and easy way to capture tasks, set due dates, and track completion status for personal productivity and daily organisation.

## Core Features

### Task Management
- Create a new task with a title and due date
- Mark an existing task as complete
- Delete a task from the list

### Task Viewing
- View all tasks displayed in a list format
- Filter tasks by completion status (complete or incomplete)
