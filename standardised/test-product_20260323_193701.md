# Simple Todo App

## Game Overview
> ⚠️ **Note:** This input describes a **productivity mobile application**, not a game. Simple Todo App is a lightweight mobile task management application for individuals who need a clean, friction-free way to organise their daily responsibilities. The core loop is simple: create a task, complete it, stay on top of your day. What makes it useful is its deliberate simplicity — no clutter, no accounts required, just tasks.

---

## Problem / Opportunity
Most task management apps (Todoist, TickTick, Notion) are over-engineered for the casual daily user. There is a clear gap for a minimal, fast, no-login task app that works immediately on first launch. The target user is an individual who wants to capture and track personal daily tasks without onboarding friction or feature overwhelm.

---

## Target Players

- **The Casual Organiser**: Age 18–45, low-to-medium tech savviness, wants to jot down tasks quickly and check them off without learning a complex tool
- **The Minimalist Power User**: Age 25–40, high tech savviness, frustrated by bloated apps, wants speed and reliability with zero noise
