# Simple Todo App

## Product Overview
A mobile application designed for individuals to manage their daily tasks efficiently. The app provides a straightforward interface for creating, tracking, and completing personal to-do items, helping users stay organised without unnecessary complexity.

## Problem Statement
Individuals often struggle to keep track of daily tasks across multiple mental lists or scattered notes. This app provides a single, simple location to capture and manage personal tasks, reducing cognitive load and helping users focus on getting things done rather than remembering what needs doing.

## Target Users
- **Individual User**: Needs a quick, frictionless way to capture tasks as they arise and track progress throughout the day. Uses the product to maintain a clear view of what needs to be done and gain satisfaction from completing items.

## Core Features

### Task Management
- Create a new task with a title and optional due date
- Mark an existing task as complete
- Delete a task from the list

### Task Viewing
- View all tasks in a scrollable list format
- Filter task list to show only incomplete tasks
- Filter task list to show only completed tasks
