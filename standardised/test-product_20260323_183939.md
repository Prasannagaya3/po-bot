# Simple Todo App

## Product Overview
A mobile application that allows individuals to manage their daily tasks efficiently. It provides a straightforward way to create, track, and complete personal tasks without the complexity of team collaboration features.

## Problem Statement
Individuals need a simple, focused way to keep track of daily tasks and responsibilities. Existing solutions are often bloated with features like team sharing, project management, and integrations that add unnecessary complexity for personal use. This app solves the need for a lightweight, distraction-free task management tool.

## Target Users
- **Individual User**: Needs to organise personal daily tasks, track what needs to be done, and feel a sense of accomplishment when completing items. Uses the product for personal productivity without requiring collaboration features.

## Core Features

### Task Management
- Create a new task with a title and due date
- Mark an existing task as complete
- Delete a task from the list

### Task Viewing
- View all tasks in a list format
- Filter task list to show only complete tasks
- Filter task list to show only incomplete tasks
