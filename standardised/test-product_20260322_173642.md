# Simple Todo App

## Product Overview
A mobile application designed for individuals to manage their daily tasks efficiently. The app provides a straightforward interface for creating, tracking, and completing personal to-do items, helping users stay organised without unnecessary complexity.

## Problem Statement
Individuals often struggle to keep track of daily tasks and responsibilities, leading to missed deadlines and forgotten commitments. This app provides a simple, distraction-free way to capture tasks quickly and maintain visibility of what needs to be done, without the overhead of complex project management tools.

## Target Users
- **Individual User**: Needs a quick, reliable way to capture and track personal tasks throughout the day; uses the product to stay organised and ensure nothing falls through the cracks

## Core Features

### Task Management
- Create a new task with a title and optional due date
- Mark an existing task as complete
- Delete a task from the list

### Task Viewing
- View all tasks in a single list format
- Filter task list to show only complete tasks
- Filter task list to show only incomplete tasks
