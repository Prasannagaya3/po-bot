# Simple Todo App

## Product Overview
A mobile app that helps individuals manage their daily tasks through simple task creation, completion tracking, and organisation. It solves the problem of keeping track of day-to-day to-dos without the complexity of full project management tools.

## Problem Statement
Individuals need a straightforward way to capture, track, and complete daily tasks without the overhead of complex productivity software. Existing solutions are often bloated with features that get in the way of simple task management.

## Target Users
- **Individual User**: Needs to quickly capture tasks, see what's pending, and mark items complete as they work through their day

## Core Features

### Task Management
- Create a new task with a title and optional due date
- Mark a task as complete
- Delete a task from the list
- Edit an existing task [Assumed: users will need to correct mistakes or update details]

### Task Viewing
- View all tasks in a list format
- Filter tasks to show only incomplete items
- Filter tasks to show only completed items
- Sort tasks by due date [Assumed: users will want to see upcoming deadlines first]

## Non-Functional Requirements
- Platform: Mobile (iOS and Android) [Assumed: "mobile app" implies both major platforms]
- Performance: Task list should load within 1 second for up to 100 tasks
- Data: Tasks should persist locally on device [Assumed: no cloud sync mentioned]

## Out of Scope
- Team sharing or collaboration features
- Push notification reminders
- Recurring tasks
- Categories or tags [Assumed: not mentioned, keeping scope minimal]
