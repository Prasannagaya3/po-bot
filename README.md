# PO Bot - Product Owner Automation

Converts any product document into a structured Jira backlog automatically.

## How it works
1. Drop a document into docs/ and push to GitHub
2. Claude reads it as a Senior PO
3. Structured backlog JSON appears in backlog/
4. You review and approve in the web app (Step 2)
5. Auto-pushed to Jira (Step 3)

## Supported formats
.pdf  .docx  .md  .txt
