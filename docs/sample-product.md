# Sample Product - Task Manager App

## Overview
A simple web-based task manager for small teams (2-10 people).
Users can create tasks, assign them to teammates, set due dates, and track progress.

## Users
- Team lead: assigns tasks, monitors progress
- Team member: picks up tasks, updates status

## Features

### Authentication
- Email and password login
- Password reset via email

### Tasks
- Create task with title, description, due date
- Assign to a team member
- Set priority: Low / Medium / High
- Update status: To Do / In Progress / Done

### Dashboard
- My tasks view
- Team view - all tasks across the team
- Filter by status and priority

## Out of scope
- Mobile app
- File attachments
- Time tracking
